--- layout: post title: Git through proxy date: 2016-09-06 16:04:20 UTC updated: 2016-09-06 16:04:20 UTC comments: false categories: --- I have to go through proxy at work. Time from time, I took a break and work on my personal projects which are hosted on github and thus, git on my computer needs to be configured for it.

On the terminal, run this :


HTTP_PROXY=http://proxy.site.my:80


git config --global http.proxy $HTTP_PROXY

Back at home, I need to set it back to having no proxy


git config --global --unset http.proxy

Yeah, tedious.
