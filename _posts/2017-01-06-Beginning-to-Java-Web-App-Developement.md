Java Servlet&nbsp;</b></span><br /><br />Servlet is a java program that run at the back end (server-side). Servlet is the program that <b><i>listen and response</i></b> to HTTP Request. Servlet is the one who retrieve data from the database by performing queries.<br /><br />Basically, Servlet is a java class (that extends HttpServlet class from Java API).<br />For example, MyServlet.java (which is then compiled to MyServlet.class)<br /><br /><div class="separator" style="clear: both; text-align: center;"><a href="https://1.bp.blogspot.com/-drphRhj43oI/WG9lxkPs-hI/AAAAAAAABpE/5HDQ7R2MA3MicQPAbXOfH96dSw3-bJlfgCLcB/s1600/servlet.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" height="320" src="https://1.bp.blogspot.com/-drphRhj43oI/WG9lxkPs-hI/AAAAAAAABpE/5HDQ7R2MA3MicQPAbXOfH96dSw3-bJlfgCLcB/s640/servlet.jpg" width="640" /></a></div><br /><br /><span style="font-size: large;"><b>5.
